# FIFA World Cup 2026 Predictor

Machine Learning project that predicts football match outcomes and estimates World Cup winning probabilities.

## Features

- Historical international football match data

- FIFA rankings integration

- Team performance statistics

- Random Forest classification model

- Match outcome probability prediction

- World Cup simulation

## Example Results

| Team | Winning Probability |

|------|--------------------|

| Brazil | 14.5% |

| France | 8.7% |

| Spain | 8.2% |

| Argentina | 8.0% |

| Morocco | 1.7% |
