# FIFA World Cup 2026 Predictor

Machine Learning project that predicts football match outcomes and estimates FIFA World Cup 2026 winning probabilities.

## Features

* Historical international football match data
* FIFA rankings integration
* Team performance statistics
* Random Forest classification model
* Match outcome probability prediction
* World Cup tournament simulation

## Technologies

* Python
* Pandas
* Scikit-learn

## Project Workflow

1. Load historical match data
2. Integrate FIFA rankings
3. Generate team statistics
4. Train a Random Forest model
5. Predict match outcomes
6. Simulate World Cup tournaments
7. Estimate winning probabilities for all 48 teams
